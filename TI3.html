ALL ABOUT WINDOWS SHORTCUT KEYS


The Shortcut Keys Of Windows 11

⚫ Windows key+Special keys

  • Window+Tab : All open windows and allows users to switch between those programmes.

  • Window+(,) /(;) : Open Emoji box

  • Window+Spacebar : Change system language

  • Window+ (+) : Zoom In

  • Window+ (-) : Zoom Out

  • Window+left/right arrow : Windows align instances

  • Window+ (#) : bring up app on task bar where the # corresponds with the position on the taskbar

  • Window+1,2,3,4.. : Open pinned app, programmes of taskbar

  • Window+ left,right,up,down arrow : Float active windows

  • Wimdow+Ctrl+D : Open/Add Virtual Desktop (Window+Ctrl+left/right arrow => Navigate opened virtual Desktops) & (Window+Ctrl+F4 => Close Virtual Desktop)

  • Wimdow+Ctrl+F : Search for PCs on a network in Windows.

  • Wimdow+Ctrl+L : Open live caption

  • Wimdow+Ctrl+M : Open Maifier settings

  • Wimdow+Ctrl+N : Open Narrator screen reader settings

  • Wimdow+Ctrl+O : Open On-Screen keyboard

  • Wimdow+Ctrl+Q : Open Quick Assist

  • Wimdow+Ctrl+S : Open Speech recognition

  • Wimdow+Ctrl+V : Open Sound output settings

  • Window+Shift+S : Screenshot [Prt scr sysRy = Directly access screenshot (snipping tool)]

  • Window+Shift+T : Extract text

  • Window+Alt+R : Directly open Screen Recorder

  • Window+X+U+U : Shut down PC

  • Window+X+P : Open installed apps

⚫ Shortcut keys to access MS Office

  • Ctrl+Shift+Window+Alt+W => MS Word

  • Ctrl+Shift+Window+Alt+P => MS PowerPoint

  • Ctrl+Shift+Window+Alt+X => MS Excel

  • Ctrl+Shift+Window+Alt+N => OneNote

  • Ctrl+Shift+Window+Alt+O => Outlook

  • Ctrl+Shift+Window+Alt+T => Microsoft Teams

  • Ctrl+Shift+Window+Alt+L => LinkedIn

⚫ Function Keys

  • F1 : (Window + F1 = Open Health Centre of Software) & (Fn+F1 = Open Mesia Player)

  • F2 : Rename the selected folder & (Fn+F2 = Decrease the volume)
    i.Edit the active cell in Microsoft Excel
    ii.Ctrl + F2 => Display a print preview in Microsoft Word
    iii.Alt + Ctrl + F2 => Open Document Library in Microsoft Office

  • F3 : Open searchbar & (Fn+F3 = Increase the volume)
    i.Shift + F3 => Lets you change from lowercase to uppercase to all caps in Word
    ii.Open find feature in Firefox and Chrome
    iii.Open Mission Control on an Apple computer running macOS X

  • F4 : There is no singel usage (* Alt+F4 = Close open tabs) & (* Alt+F4+Enter = Shut down pc) & (Fn+F4 = Mute the device)

  • F5 : Refresh (Fn+F5 = Play previous song/video)
    i.Ctrl + F5 : Completely refresh a web page, clearing the cache and downloads all content again
    ii.Opens Find and Replace in Microsoft Office

  • F6 : From opened browser tabs to url (Fn+F6 = Play next song/video) & (Ctrl + Shift + F6 => Lets you easily switch between Word documents)

  • F7 : (Applied Only MS word) Alt+F7 => Spelling correction & Shift + F7 => Open Thesaurus in Microsoft Word
    & (Fn+F7 = Play/Pause song/video)

  • F8 : (For MS word) Select all
    * (For booting time) Turn on Save mode
    [i.Switching Display Modes:
    On some laptops, pressing Fn + F8 allows you to switch between different display modes. For example: LCD Screen: If you’re using an external monitor or projector, this combination may switch the display back to your laptop’s built-in screen. External Monitor/TV: If you’re using your laptop screen, it might switch the display to an external monitor or TV. This feature is especially handy for presentations or when connecting your laptop to an external display.
    ii.Multimedia Control:
    In certain programs (such as media players), Fn + F8 can skip to the next audio track or song. For instance, if you’re using Windows Media Player, pressing this combination will move to the next track in your playlist.
    iii.Browser Functionality:
    In web browsers, Fn + F8 might have a different function. For example: It could minimize or maximize the browser window. It might perform other browser-specific actions.]

  • F9 : There is no singel usage
    (In MS word =>
    * Ctrl+F9 = Create a new field code formatting marks. It is particularly useful for tasks like updating fields, such as in a Table of Contents, within a Word document.
    * Alt+F9 = Hide and show field codes within the document. It is particularly useful for displaying or concealing the field codes that are used in Word documents.
    * shift+F9 = Switch between a field code and its result.This function is useful for displaying or concealing the field codes that are used in Word documents.
    In Excel =>

    * Alt+F9 = Recalculates all formulas in the workbook
    * Shift+F9 = Recalculates only the active worksheet and its dependencies.
    * Ctrl+F9 = Calculates all worksheets in all open workbooks, regardless of whether they have changed since the last calculation. )

  • F10 : There is no singel usage
    ( In MS WORD, MS POWERPOINT, MS EXCEL,NotePad,WordPad, MS Paint => To show keytips / shortcut keys
    In MS excel =>
    Shift+F10 : Use to left alignment a paragraph Alt+F10 : Hide and show the Selectionpane)

  • F11 : Directly, Full Screen mode is on/off (if there is any ) & Shift + F11 => Add a new spreadsheet in Excel

  • F12 : i. Save file in MS word
    ii. Reboot Dell laptop / desktop
    iii.Shift + F12 => Save Word document
    iv.Ctrl + F12 => Open Word document
    v.Ctrl + Shift + F12 => Print Word document
