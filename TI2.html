ALL ABOUT WINDOWS SHORTCUT KEYS


The Shortcut Keys Of Windows 11

⚫ Ctrl+A to Ctrl+Z

  • Ctrl+A : All select

  • Ctrl+B : Bold

  • Ctrl+C : Copy

  • Ctrl+D : Font,Duplicate

  • Ctrl+E : Center

  • Ctrl+F : Find

  • Ctrl+G : Go to

  • Ctrl+H : Replace

  • Ctrl+I : Italic

  • Ctrl+J : Justify

  • Ctrl+K : Hyperlink

  • Ctrl+L : Align text to left

  • Ctrl+M : Hanging Indent

  • Ctrl+N : New page/document

  • Ctrl+O : Open

  • Ctrl+P : Print

  • Ctrl+Q : Add space after paragraph

  • Ctrl+R : Align text to right

  • Ctrl+S : Save

  • Ctrl+T : Left Indent

  • Ctrl+U : Underline

  • Ctrl+V : Paste

  • Ctrl+W : Close opened tabs/programmes

  • Ctrl+X : Cut

  • Ctrl+Y : Repeat/Redo

  • Ctrl+Z : Undo

⚫ Windows key+A to Windows key+Z

  • Window+A : Open action center

  • Window+B : Set focus on the notification area(right side of the Taskbar)

  • Window+C : (For Windows 10 users) Open Cortana & (For Windows 11 users) Open Copilot

  • Window+D : Show/Hide Desktop

  • Window+E : Open File Explorer

  • Window+F : Feedback Hub

  • Window+G : Access the recording in Xbox Record screen

  • Window+H : Speech to text

  • Window+I : Open display settings

  • Window+J : No Use

  • Window+K : Connect a cable to cast

  • Window+L : Lock screen

  • Window+M : Minimize all opened tabs, programmes (Window+Shift+M : Maximise the minimized tabs)

  • Window+N : Open notification panel

  • Window+O : Change Orientation(In MS office) (But WIndows 11 users can't use it)

  • Window+P : Presentation display

  • Window+Q : Open Searchbar ( in Windows11)

  • Window+R : Open Run command prompts

  • Window+S : Open Searchbar

  • Window+T : Jump on the taskbar

  • Window+U : Open display settings

  • Window+V : Open Clipboard

  • Window+W : Open Widgets panel (windows10: open whiteboard & full screen snip)

  • Window+X : Open quick line menu

  • Window+Y : Open Yahoo mail (If there is any Yahoo mail in user's account)

  • Window+Z : Create Frame for multitasking
