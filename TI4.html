ALL ABOUT WINDOWS SHORTCUT KEYS


The Shortcut Keys Of Windows 11

⚫ Extra Shortcut Keys

* In MS excel :-

  • Ctrl+Spacebar : Column Select

  • Shift + Spacebar : Row Select


*In Desktop :-

  • Ctrl + Alt + Delete : Switch Account (if there are many user accounts in one desktop)

  • Ctrl+Alt+F : Full screen zoom

  • Ctrl +Shift +T : Restore tab/Undo the deleted tab

  • Ctrl+Shift+Esc : Directly open task manager

  • Ctrl+Tab : To move text

  • Alt+ Spacebar : Open a search box due to the PowerToys Run feature.( If there is no PowerToy service , the shortcut feture will not run)

  • Ctrl+Shift+Tab : Move to previous tab

  • Alt+Left/Right arrow : Navigate the webpages to Forward / Backward.

  • Shift+Delete : Totally delete (not save in recycle bin)

  • Shift+ up,down,left,right arrow : Select text (without use the mouse)

  • Shift+Ctrl+Up,Down,Left,Right arrow : Select entire words

  • Shift+Ctrl+Left arrow : Select anything to left

  • Shift+Ctrl+Right arrow : Select anything to right

  • Ctrl+Delete : Delete the character
  • Ctrl+Backspace : Delete the character
  • The Backspace key erases the text to the left of the insertion point one character at a time. On the other hand , the Delete key erases the text to the right of the insertion point.

  • Alt+Enter key : Open propertise of selected file/folders

  • Ctrl+Left click : Select multiple individual items

  • Shift+Left click : Select multiple items

  • Alt+Shift+Up/Down arrow : 1.Quickly Adjust Paragraph Order
    2.Quickly Adjust Table Row Order
    3.Quickly Split and Merge Tables
    4.Quickly Adjust the Outline Level of a Paragraph

  • Ctrl+F10 : Maximize or restore the selected workbook window (In MS Excel)

  • Ctrl+Scroll button(up scrolling) : Zoom In the screen
  • Ctrl+Scroll button(down scrolling) : Zoom Out the screen




* In Web browser :-

  • Ctrl+H : Open web browser's history

  • Ctrl+Shift+Delete : Delete web browser's history

  • Ctrl+J : Download apps in Web browser (Non of those installed app in the system)

  • Ctrl +Shift +N : Open Incognito mode

    But in the desktop , file explorer (Ctrl+Shift +N => Create a new folder)
